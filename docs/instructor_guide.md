# Instructor Facilitation Guide — Python for Financial Analysts

This guide supports facilitators delivering the GeekInk Python for Finance Bootcamp.

---

# Teaching Philosophy

- Practical-first learning  
- Real datasets  
- Gradual complexity  
- Community collaboration  
- High encouragement and participation  
- Consistent feedback  

---

# Weekly Teaching Breakdown

## Week 1 — Python Core
- Introduce variables, loops, functions  
- Show real-world examples: ROI, interest, profit margins  
- Ensure every student runs a full script  
- Assign Week 1 task  

## Week 2 — Analysis & Visualization
- Introduce Pandas and Matplotlib  
- Live-clean a CSV dataset  
- Teach moving averages & returns  
- Visualize stock trends  
- Assign Week 2 task  

## Week 3 — Modeling & Forecasting
- Introduce linear regression for finance  
- Explain correlation and risk  
- Build portfolio metrics  
- Help students select capstone projects  

---

# Assessment Structure
- **40% – Weekly assignments**  
- **60% – Capstone project**  
- Participation & engagement encouraged  

---

# Instructor Checklist

✔ Python installed  
✔ Jupyter Notebook running  
✔ Students understand CSV import  
✔ Students create at least 4 charts  
✔ Capstone PR submitted
