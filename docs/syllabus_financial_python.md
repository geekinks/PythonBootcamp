# Python for Financial Analysts — 3-Week Syllabus (GeekInk)

## Overview
This course teaches Python as a practical tool for financial analysis, reporting, forecasting, and portfolio modeling.

Hands-on, beginner-friendly, and dataset-driven.

---

# Week 1 — Core Python

### Topics
- Python setup  
- Variables, data types  
- Lists, dicts, loops  
- Functions  
- Reading/writing CSV  
- Profit, ROI, interest calculations  

### Assignment
Build a script that reads sales CSV → calculates KPIs → exports summary.

---

# Week 2 — Data Analysis & Visualization

### Topics
- Pandas fundamentals  
- Cleaning data  
- Aggregation / correlation  
- Stock price analysis  
- Moving averages  
- Volatility  
- Matplotlib & mplfinance  

### Assignment
Analyze a real asset’s dataset and produce:
- Cleaned data  
- Moving averages  
- Daily returns  
- Trend charts  

---

# Week 3 — Forecasting & Portfolio Modeling

### Topics
- Time series forecasting  
- Linear regression  
- Correlation matrix  
- Risk vs return  
- Sharpe ratio  
- Mini-project development  

### Capstone
Choose one:
1. Portfolio Analyzer  
2. Price Forecasting Model  
3. Automated Financial Dashboard
