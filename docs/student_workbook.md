# Student Workbook — Python for Financial Analysts

This workbook accompanies the 3-week Python for Finance Bootcamp. It contains practice notes, exercises, reflection prompts, and learning checkpoints.

---

# Week 1 — Core Python Foundations

### Key Concepts
- Variables, data types  
- Lists, tuples, dictionaries  
- Loops & conditionals  
- Functions & modular code  
- Reading/Writing CSV  
- Basic financial arithmetic

### Quick Exercise
Write a Python script that:
1. Reads `sales_data.csv`
2. Computes:
   - Revenue  
   - Cost  
   - Profit  
   - Profit Margin  
3. Saves the summary back to CSV.

### Self-Check Questions
- What is the difference between a list and a dictionary?
- What does `return` do in a function?
